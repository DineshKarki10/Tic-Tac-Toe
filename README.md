# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against a friend in this simple yet engaging web-based game!

## 🎮 Features

- **Two-Player Gameplay**: Take turns with a friend to play X and O
- **Win Detection**: Automatically detects and announces the winner
- **Draw Detection**: Recognizes when the game ends in a draw
- **Reset Functionality**: Start a new game anytime with the reset button
- **Responsive Design**: Works on different screen sizes
- **Clean UI**: Modern and intuitive user interface

## 🎯 How to Play

1. The game starts with Player O's turn
2. Click on any empty box to place your mark (O or X)
3. Players alternate turns
4. Get three of your marks in a row (horizontally, vertically, or diagonally) to win
5. If all boxes are filled without a winner, it's a draw
6. Click "Reset Game" or "Play Again" to start over

## 🛠️ Technologies Used

- **HTML5**: Structure and markup
- **CSS3**: Styling and responsive design
- **JavaScript (ES6)**: Game logic and interactivity

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DineshKarki10/Tic-Tac-Toe.git
   ```

2. Navigate to the project directory:
   ```bash
   cd tic-tac-toe
   ```

3. Open `tictactoe.html` in your web browser

That's it! No additional dependencies or server setup required.
